## DNS 和 HTTPDNS  

### DNS

用户与互联网中的某台主机通信的时，必须知道对方的 IP 地址，然而用户很难记住长达 32 位的 IP 二进制主机地址，即使是点分十进制的 IP 地址也是不太容易记忆。为了方便记忆，引入了域名系统，用来便于人们使用的机器名字转换成 IP 地址。  

域名和 IP 地址的映射关系就是记录在 DNS 服务器中的。
